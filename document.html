Introduction

Python is a high-level, interpreted programming language created by Guido van Rossum and released in 1991.

Python's design philosophy emphasizes code readability with its notable use of significant whitespace.

It provides constructs that enable clear programming on both small and large scales.

  • Easy to learn and read
  • Large standard library
  • Dynamic type system
  • Automatic memory management
  • Multiple programming paradigms
Getting Started

To begin using Python, first download and install it from python.org.

After installation, you can verify Python is installed by opening a terminal and typing:

python --version

Create your first Python program by saving this code in a .py file:

print("Hello, World!")

Run the program from the terminal:

python hello.py
Python Syntax

Python uses indentation to define code blocks, instead of curly braces or keywords.

Indentation is mandatory in Python code.

if True: print("True") if True: print("Nested")

Python statements often don't need semicolons to indicate end of line.

Multi-line statements use backslash or parentheses:

total = 1 + \ 2 + \ 3 # or total = (1 + 2 + 3)
Data Types

Python has several built-in data types:

# Numbers x = 5 # int y = 3.14 # float z = 1 + 2j # complex # Strings name = "Python" multiline = '''Multiple line string''' # Lists numbers = [1, 2, 3, 4, 5] # Tuples coordinates = (10, 20) # Dictionaries person = {"name": "John", "age": 30} # Sets unique_numbers = {1, 2, 3}

Variables are dynamically typed:

x = 5 # x is an integer x = "Hello" # now x is a string

Type conversion functions:

str(123) # Convert to string int("123") # Convert to integer float("3.14") # Convert to float
Control Flow

Control flow is handled using if statements, loops, and conditional expressions.

If statements:

age = 18 if age < 13: print("Child") elif age < 20: print("Teenager") else: print("Adult")

For loops:

for i in range(5): print(i) for item in ["apple", "banana", "orange"]: print(item)

While loops:

count = 0 while count < 5: print(count) count += 1
Functions

Functions are defined using the def keyword:

def greet(name): return f"Hello, {name}!" print(greet("Python"))

Functions can have default parameters:

def power(x, exponent=2): return x ** exponent print(power(2)) # 4 print(power(2, 3)) # 8

Lambda functions for simple operations:

square = lambda x: x ** 2 print(square(5)) # 25
Object Oriented Programming

Python is an object-oriented programming language.

Classes are defined using the class keyword:

class Person: def __init__(self, name, age): self.name = name self.age = age def greet(self): return f"Hello, I'm {self.name}" person = Person("Alice", 30) print(person.greet())

Inheritance is supported:

class Student(Person): def __init__(self, name, age, grade): super().__init__(name, age) self.grade = grade
Modules and Packages

Python code can be organized into modules and packages.

Import statements are used to include modules:

import math print(math.pi) from datetime import datetime print(datetime.now()) import random as rd print(rd.randint(1, 6))

Creating your own module:

# mymodule.py def greet(name): return f"Hello, {name}!" # main.py import mymodule print(mymodule.greet("Python"))
Error Handling

Python uses try/except blocks for error handling:

try: x = int("abc") except ValueError: print("Invalid conversion") except Exception as e: print(f"Error: {e}") else: print("No error occurred") finally: print("This always runs")

Raising exceptions:

def divide(a, b): if b == 0: raise ValueError("Cannot divide by zero") return a / b
File Operations

Python can read from and write to files:

# Writing to a file with open("file.txt", "w") as f: f.write("Hello, World!") # Reading from a file with open("file.txt", "r") as f: content = f.read() print(content)

Working with CSV files:

import csv with open("data.csv", "w") as f: writer = csv.writer(f) writer.writerow(["Name", "Age"]) writer.writerow(["Alice", 30])

File modes include:

  • "r" - Read (default)
  • "w" - Write (overwrites)
  • "a" - Append
  • "b" - Binary mode
